Description

An RDF vocabulary for describing members of a federation to be considered for querying by the HeFQUIN engine.

Interface c

IRI http://w3id.org/hefquin/feddesc#Interface
Description

An interface is a means to issue data retrieval requests to a federation member. This class is a superclass of any kind of interface and should not be used directly within federation descriptions. Concrete types of interfaces, to be used in federation descriptions, are captured by the subclasses of this class.

This class is a superclass of any kind of fragment interface and should not be used directly within federation descriptions.
